What is Git?

Its a tool that tracks changes in code as and how you progress with revisions and edits. 
We can see what has changed in the code and collaborate with others without overwriting eachothers work.

How to Push Code to GitHub?


### Step 1 — Create a repository on GitHub

1. Go to https://github.com and sign in
2. Click the **+** button then click **New repository**
3. Give it a name (e.g. `my-first-project`) and click **Create repository**

### Step 2 — Connect your local folder to GitHub

Navigate to your project folder:

add origin https://github.com/your-username/my-first-project

### Step 3 — Stage, commit, and push your code

Commands are the heart of Git and feels very linux-ish to use as it was meant to be used in the terminal. Commands are run every time you want to save your work to GitHub.

Commit changes is to be done every time you make a change to anything in the repo. 


## What is Markdown?

Markdown is a simple way to format text using plain characters. Instead of clicking a **Bold** button like in Word, you just wrap text in `**asterisks**`.

It's used everywhere — GitHub, Discord, Reddit, README files, blogs, etc.


## Basic Markdown Syntax

### Headings

Put `#` symbols at the start of a line to make headings. More `#` = smaller heading.


# Heading 1  (biggest)
## Heading 2
### Heading 3  (smaller)




### Lists

**Unordered list** (bullet points) — use `-` or `*`:

- Apples
- Bananas
- Mangoes

**Ordered list** (numbered) — just use numbers:

1. Install Git
2. Create a GitHub account
3. Push your first commit
